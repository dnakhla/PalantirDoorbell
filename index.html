Because Your Doorbell Wasn't Creepy Enough

I never check my doorbell camera - but what if an AI agent could watch where I wouldn't? This project captures photos of people passing by and creates comprehensive profiles using 2 streamlined AI analyses per person. 1. Person Detection & Capture

YOLO v8 monitors 1920x1080 RTSP stream with 80% confidence threshold. When a person is detected, system captures full-frame image with bounding box and saves original high-quality photo to disk. This saves a still image of everyone who passes the house.

Because humans can't monitor video feeds 24/7 but AI can

2. AI Pre-screening

GPT-4.1-mini receives the captured image and determines if person is clearly identifiable. Returns confidence level and description. If person isn't recognizable, image is deleted. If identifiable, proceeds to comprehensive analysis.

Quality control before the real profiling begins

3. Comprehensive Analysis

Single GPT-4.1-mini call with context analyzes: nickname, physical description, clothing, purpose, demeanor, notable features, and 2-3 sentence summary. Assumes normal neighborhood activities, no "suspicious" behavior nonsense.

Turning computer vision into friendly neighborhood intelligence

4. Profile Creation & Timeline

System creates/updates person profile with AI analysis, stores image paths, and updates timeline dashboard. Each person gets nickname, visit count, and comprehensive analysis visible on hover. All data persists locally.
